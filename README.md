# QtSDK-WindowsDocker

## 中文

一個使用 Qt SDK 線上安裝程式建立 Docker Image 的範例。

使用者必須自行更換 `qtaccount.ini` 這個檔案，已完成自動登入。

## English

A sample to build docker image with Qt SDK online installer.

User need to replace the file `qtaccount.ini` for automatic login.
